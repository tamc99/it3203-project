About This Project

Exploring Web Server Technology Evolution

Project Overview

This website is a research project created for IT 3203 Web Development at Kennesaw State University during Fall Semester 2025. The project explores the evolution of web server technology from Tim Berners-Lee's original CERN httpd server in 1990 through the modern era of cloud-native platforms.

Research Focus

Our research examines how web servers transformed from simple document-sharing tools into sophisticated application platforms capable of handling millions of concurrent users. We trace this evolution through three major eras:

  • The Foundation Era (1990-1995) featuring CERN httpd and NCSA HTTPd
  • The Open Source Revolution (1995-2004) dominated by Apache HTTP Server
  • The Modern Era (2004-present) characterized by event-driven architecture and Nginx

Project Goals

This project aims to help students and web development learners understand the technical innovations, architectural decisions, and historical context that shaped modern web infrastructure. By understanding how and why web servers evolved, developers can make better decisions about server selection, configuration, and deployment strategies for contemporary applications.

This is a class project for IT 3203 Web Development

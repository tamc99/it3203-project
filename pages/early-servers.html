Foundation Era: Early Web Servers

CERN httpd and NCSA HTTPd (1990-1995)

The Birth of the Web Server

The first generation of web server established the fundamental client-server interaction that we still see today. These servers laid the groundwork for the entire World Wide Web we know today, transforming how humans share and access information.

CERN httpd: The First Web Server (1990)

Tim Berners-Lee's Vision

According to CERN's official history, Tim Berners-Lee created the first web server, CERN httpd, in 1990 as part of his broader vision for a "web of human knowledge." This server was designed to specifically share scientific documents between researchers at CERN and other institutions around the world.
